# da-leclerc.github.io

Bilingual (EN/FR) professional website for Dominic-André Leclerc — AI Integration & Compliance Advisor.

Built with [Hugo](https://gohugo.io/) and the [PaperMod](https://github.com/adnanh/hugo-PaperMod) theme. Deployed via GitHub Pages.

## Local Development

```bash
# Clone with submodules
git clone --recurse-submodules https://github.com/DA-Leclerc/da-leclerc.github.io.git
cd da-leclerc.github.io

# Run local server
hugo server -D
```

## Deployment

Pushes to `main` automatically trigger a GitHub Actions build and deploy to GitHub Pages.
